### 如何将我的插件添加到这?

我们十分欢迎您将相关的**KotoriBot**插件添加至此，只需一个简单的**Pull Request**即可完成添加，当然，记得手动一下完善上面的插件列表项(添加至最前面)，你也可以将你插件的说明/介绍/自述文件添加至你的插件文件夹根目录

插件文件夹根目录:统一使用英文小写,除了可使用"-"连接多个单词以外不可使用其它非英文和符号
